# Encode and decode 'Sqids'

Encode and decode 'Sqids'

## Usage

``` r
sqids_encode(numbers, salt)

sqids_decode(ids)
```

## Arguments

- numbers:

  A list of integers.

- salt:

  An integer vector.

- ids:

  A character vector.

## Value

For `sqids_encode()`, a list of integers. For `sqids_decode()`, a
character vector.
