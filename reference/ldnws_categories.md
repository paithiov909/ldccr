# List of categories of the Livedoor News Corpus

List of categories of the Livedoor News Corpus

## Usage

``` r
ldnws_categories(
  keep = c("dokujo-tsushin", "it-life-hack", "kaden-channel", "livedoor-homme",
    "movie-enter", "peachy", "smax", "sports-watch", "topic-news")
)
```

## Arguments

- keep:

  Category names to parse.

## Value

A character vector.
