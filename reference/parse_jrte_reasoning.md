# Parse reasoning column of 'rte.\*.tsv'

Parse reasoning column of 'rte.\*.tsv'

## Usage

``` r
parse_jrte_reasoning(tbl)
```

## Arguments

- tbl:

  A tibble returned from
  [`read_jrte()`](https://paithiov909.github.io/ldccr/reference/read_jrte.md)
  of which name is `rte.*.tsv`.

## Value

A tibble.
